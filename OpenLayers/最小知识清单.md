### 🧭 **OpenLayers 最小知识清单**

**只记这些，你就能读懂 90% 场景，并在 AI 帮助下轻松写代码。**

```apl
1. new Map({...}) 创建地图，target 是容器 div。
2. View 控制中心点、缩放级别、投影。
3. Layer = 图层；最常用的是 TileLayer（底图）和 VectorLayer（要素层）。
4. VectorSource 管数据；VectorLayer 只是显示。
5. Feature 是地图上的元素（点、线、多边形）。
6. geometry: new Point([lng, lat]) 添加一个点。
7. Style 用来控制图标、颜色、线宽等外观。
8. Icon 用于给点设置图片。
9. coords 必须是投影坐标（常用的是 EPSG:3857）。
10. fromLonLat([lng, lat]) 将经纬度转换为地图坐标。
11. map.addLayer(layer) 将图层加到地图上。
12. layer.getSource().addFeature(feature) 添加元素。
13. map.on('click', fn) 监听地图点击事件。
14. map.forEachFeatureAtPixel(e.pixel, fn) 判断是否点中了某个 feature。
15. setStyle() 可以给某个 feature 单独换样式。
16. Overlay 用来做气泡、弹窗 HTML。
17. interaction 决定用户怎么跟地图交互（缩放、拖拽等）。
18. controls 是 UI（缩放按钮、比例尺等）。
19. removeFeature / clear 清除标记。
20. 地图所有东西都是：Map → Layer → Source → Feature 这条链。
```
